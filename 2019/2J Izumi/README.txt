2J 泉 知成 映像認識防犯システム(仮)について

・現状
　　ラズパイの外付けカメラから得た映像から人の顔を認識することができた(精度いまいち)
　　特定のキーを押したときにのみ写真を撮影し、その画像データをLINEに送信する(←今ここ)

本当は「顔認証」や「表情からの感情認識」等をやりたかったが、高専プロコン(競技部門)の関係や、その他学校の課題・テスト勉強などで忙しく想定していたよりも開発することができなかった。

・将来的に実装したい機能
　　顔認証でその顔が誰なのかを特定する。
　　表情から感情を読み取る。

とりあえずオープンキャンパスまでにはLINEの送信機能までを確実に実装するようにする。
オープンキャンパス以降は高専プロコンに使用する遺伝的アルゴリズムを優先して作成するが、夏休みなど時間が空いているときに開発したいと思う。(できれば高専祭までに...)
